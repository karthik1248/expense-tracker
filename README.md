# Expense Tracker

A full-stack Expense Tracker web application.

## Tech Stack

Frontend:
- React
- Tailwind CSS
- Vite

Backend:
- Node.js
- Express.js
- MongoDB

---

# Setup Instructions

## Clone Repository

```bash
git clone https://github.com/karthik1248/expense-tracker
```

---

## Install Frontend

```bash
cd client
npm install
```

---

## Install Backend

```bash
cd server
npm install
```

---

## Create .env File

Inside `server/` create a `.env` file and add:

```env
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret
PORT=3000
```

---

## Run Backend

```bash
npm run dev
```

---

## Run Frontend

```bash
npm run dev
```
